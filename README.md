# 🌿 WellBot — AI-Powered Wellness Platform

> A full-stack wellness management application that combines daily health tracking, AI-generated nutrition plans, stress monitoring, and personalised mental health recommendations — all in one beautifully designed dashboard.

---

## 📸 Overview

WellBot helps users take charge of their physical and mental wellbeing through:

- **Daily Check-Ins** — Log mood, hydration, energy, sleep and daily goals
- **AI Nutrition Plans** — Personalised meal plans generated by Google Gemini based on your BMI and diet preference
- **Stress Tracker** — Log and monitor stress levels over time with AI-generated coping protocols
- **Health Reports** — View wellness score history and export PDF health summaries
- **AI Chatbot (WellBot)** — An intelligent wellness assistant powered by Gemini
- **Recommendations** — Dynamic, personalised stress management protocols
- **Feedback** — Submit and view platform feedback with rating analytics

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- Python 3.11+
- A [Supabase](https://supabase.com) project
- A [Google AI Studio](https://aistudio.google.com) API key

---

### 1. Clone the Repository

```bash
git clone <your-repo-url>
cd frontend-infosys_m
```

---

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file inside the `backend/` directory:

```env
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_SERVICE_KEY=your-service-role-key
GOOGLE_API_KEY=your-google-gemini-api-key
```

Start the backend server:

```bash
uvicorn app.main:app --reload --port 8000
```

The API will be available at: `http://localhost:8000`

---

### 3. Frontend Setup

```bash
cd project
npm install
```

Create a `.env` file inside the `project/` directory:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

Start the development server:

```bash
npm run dev
```

The app will be available at: `http://localhost:5173`

---

Run the following SQL migration scripts in your Supabase SQL editor:

## ✨ Features

### 🏠 Overview Dashboard
- Real-time wellness score calculated from check-ins
- Weekly wellness score bar chart
- Mood distribution breakdown
- Average hydration tracking
- Platform-wide user stats & feedback score

### 📋 Daily Check-In
- Log mood (Great / Good / Okay / Low / Bad)
- Track daily water intake (glasses)
- Energy & sleep sliders
- Interactive goal checklist (Morning meditation, hydration, exercise, etc.)
- Data persists and pre-fills on page reload

### 🥗 Nutrition
- BMI calculator with interactive weight/height controls
- Diet preference selector (Vegetarian / Non-Vegetarian)
- AI-generated personalised meal plan via Google Gemini
- Macro breakdown (Protein / Carbs / Fats)
- Hydration tracker synced with daily check-ins

### 😰 Stress Tracker
- Log daily stress levels
- AI-generated personalised stress management protocols
- Real-time stress trend analysis

### 📊 Health Reports
- 6-month wellness score history chart (computed from real check-in data)
- Filter by category: Wellness / Stress / Nutrition / Sleep
- Export a personalised PDF report with real user data
- Wellness score algorithm: Mood (30%) + Energy (30%) + Sleep (25%) + Stress (15%)

### 🤖 WellBot Chat
- Conversational AI wellness assistant
- Powered by Google Gemini
- In-app chat widget accessible from any page

### 🎯 Recommendations
- Dynamic stress management protocols generated by Gemini
- Personalised based on user's current stress score

### ⭐ Feedback
- Submit star ratings and written feedback
- View aggregated platform rating statistics

## 🔐 Authentication

WellBot uses [Supabase Auth](https://supabase.com/docs/guides/auth) with JWT tokens:

- Sign Up / Sign In with email & password
- All API requests are authenticated via `Bearer` token
- Backend validates tokens using the Supabase service key
- `fetchWithAuth()` utility automatically attaches the session token to every API call

## 🌐 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/daily-checkins/` | Create or update today's check-in |
| `GET` | `/daily-checkins/` | List check-ins (with date range filter) |
| `POST` | `/nutrition/goals` | Save BMI and diet goals |
| `GET` | `/nutrition/goals` | Fetch nutrition goals |
| `GET` | `/nutrition/meal-plan` | Get or regenerate AI meal plan |
| `POST` | `/stress-logs/` | Log a stress entry |
| `GET` | `/stress-logs/` | List recent stress logs |
| `GET` | `/reports/list` | Get report cards with real scores |
| `GET` | `/reports/history` | Get 6-month wellness score history |
| `GET` | `/reports/download/{id}` | Download a PDF health report |
| `POST` | `/wellbot/chat` | Send a message to the AI chatbot |
| `GET` | `/recommendations/` | Get AI stress protocol recommendations |
| `POST` | `/feedback/` | Submit user feedback |
| `GET` | `/feedback/stats` | Get platform feedback statistics |
| `GET` | `/profiles/count` | Get total registered user count |
